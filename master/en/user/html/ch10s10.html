10.10.1. About predictors

Predictors are data elements automatically generated from samples of current and past data. The most immediate application of predictors is in the simplification of monitoring or surveillance rules which allows them to be specified as simple validation rules. A predictor specifies an expression and a set of *sample constraints* (identical to the current implementation of validation rules). This includes a period type, organisation unit levels, and parameters specifying past or periodic sample windows.

You use predictors to analyse previous data to predict future values, for example for disease surveillance.

10.10.2. Create a predictor

  1. Open the Maintenance app and click Other > Predictor.

  2. Click the add button.

  3. In the Name field, type the predictor name.

  4. (Optional) In the Code field, assign a code.

  5. (Optional) Type a description.

  6. Select a Period type.

  7. Assign organisation unit levels.

    Tip

    If you're missing organisation unit level, you can create new ones.

  8. Create a Generator:

  9. Click Sample skip test.

  10. Sequential sample count

  11. Annual sample count

  12. Sequential skip count

  13. Click Save.

10.10.3. Edit predictors

  1. Open the Maintenance app and click Other > Predictor.

  2. In the list, find the predictor you want to modify.

  3. Click the options menu and select Edit.

  4. Modify the options you want.

  5. Click Save.

10.10.4. Clone predictors

  1. Open the Maintenance app and click Other > Predictor.

  2. In the list, find the predictor you want to clone.

  3. Click the options menu and select Clone.

  4. Modify the options you want.

  5. Click Save.

10.10.5. Delete predictors

  1. Open the Maintenance app and click Other > Predictor.

  2. In the list, find the predictor you want to delete.

  3. Click the options menu and select Delete.

  4. Click Confirm.

10.10.6. Display details of predictors

  1. Open the Maintenance app and click Other > Predictor.

  2. In the list, find the predictor you want to view.

  3. Click the options menu and select Show details.

10.10.7. Translate predictors

  1. Open the Maintenance app and click Other > Predictor.

  2. In the list, find the predictor you want to translate.

  3. Click the options menu and select Translate.

  4. Select a locale.

  5. Type a Name, Short name and Description.

  6. Click Save.
